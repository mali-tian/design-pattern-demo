# Demos of Design Pattern

Java version: 11.0.1
gradle version: 6.0.1
gradle wrapper: `gradle wrapper --gradle-version 6.0.1`


#### Run Test

`./gradlew test`
Check test report under `./build/reports/tests/test/index.html`


#### Build Jar

`./gradlew build`
Build result under `./build/libs/`
